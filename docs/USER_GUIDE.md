## Overview

TalentOS is a multi-tenant PaaS for modelling agencies, pageant organizers, and talent businesses.

## Dashboards

| Role | Dashboard | Key features |
|------|-----------|--------------|
| **Modelling** | Modelling dashboard | Bookings, talent roster, castings |
| **Admin** | Tenant admin | Users, roles, finance, compliance |
| **Pageant** | Pageant dashboard | Events, registrations, judges, results |
| **Academy** | Academy dashboard | Courses, cohorts, certificates |
| **Influencer** | Influencer dashboard | Campaigns, creators, deliverables |
| **Super Admin** | Platform admin | Tenants, features, governance |

## Getting started

1. **Login** – Use credentials provided by your tenant admin
2. **Tenant switcher** – Select your organization (admin users)
3. **Navigation** – Use the sidebar to access modules

## Support

- In-app: Support cases (Super Admin)
